# Global CO2 Emissions & Economic Growth Analysis (1980-2023) 🌍📈

## 📌 Project Overview
This project was developed for the **Leeds DATA AI Hackathon**. It focuses on analyzing the relationship between **CO2 emissions**, **GDP (Economic Growth)**, and **Population** across different countries and regions from 1980 to 2023. 

The core objective is to visualize **Decoupling**—the phenomenon where an economy continues to grow while its carbon emissions stabilize or decline—providing insights into global sustainable development.

## 🚀 Key Features
* **Data Integration:** Merging multiple datasets (CO2, GDP, Population) into a unified master table.
* **Decoupling Analysis:** Identifying trends where economic growth and environmental impact diverge.
* **Interactive Visualizations:** * Dynamic **Bar Chart Race** showing global CO2 ranking shifts over 40+ years.
    * Interactive **Decoupling Analysis** charts created with Plotly.
